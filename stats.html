SharePoint GET Endpoint

To load analytics from SharePoint, create a second Power Automate flow with an "When an HTTP request is received" trigger (GET method) that queries your SharePoint list and returns the items as JSON. Paste the flow URL below.

How to Create the GET Flow

  1. Go to make.powerautomate.com
  2. Create → Instant cloud flow → Skip
  3. Add trigger: "When an HTTP request is received"
  4. Set Method to GET
  5. Add action: "Get items" from your SharePoint list
  6. Add action: "Response" → Status 200 → Body = output of Get items
  7. Add header: Access-Control-Allow-Origin: *
  8. Save → copy the HTTP GET URL → paste above
