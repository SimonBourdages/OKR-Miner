# OKR Miner

### ⛏️ Engine Building Game

A space-themed idle/engine building game where you place automation buildings on a cosmic canvas to generate OKRs. Built as a single HTML file with vanilla JavaScript and HTML5 Canvas.

🎮 **Play now:** [game.thisai.fun](https://game.thisai.fun)
📱 **Android APK:** [Download from Releases](../../releases/tag/android-latest)

---

## 🚀 How to Play

1. Open `index.html` in a modern browser (or visit the live URL above)
2. Click **START GAME** from the main menu
3. Select a building type from the right-side panel (desktop) or bottom panel (mobile)
4. Click on the canvas to place buildings — they automatically generate OKRs
5. Reach OKR milestones (OKR Reviews) to unlock new building types and upgrades
6. Click existing buildings to upgrade them for higher output
7. Keep building your automation empire!

## 🏗️ Building Types

| Building | Cost | Base Output | Description |
|---|---|---|---|
| **Collector** | 50 AC | 1 OKR/s | Basic gathering node — your starting building |
| **Processor** | 150 AC | 5 OKR/s | Processes raw OKRs at higher throughput |
| **Multiplier** | 500 AC | 25 OKR/s | Amplifies output from nearby buildings |
| **Generator** | 1500 AC | 80 OKR/s | High-output power generation node |
| **Bank** | 5000 AC | 250 OKR/s | Stores and compounds OKR interest |
| **Synthesizer** | 15000 AC | 1000 OKR/s | End-game building with massive output |

## ⚙️ Game Mechanics

- **Milestones (OKR Reviews):** Reach OKR thresholds to advance through reviews, unlocking buildings and features
- **Electricity:** Powers your buildings — manage supply to keep everything running
- **Spaceships:** Automated transports that move OKRs between buildings and the factory
- **Repair Bots:** Keep your buildings maintained and running at peak efficiency
- **Themes:** Unlock visual themes as you progress (Emerald, Crimson, Ocean, and more)
- **Events:** Random space events that can boost or challenge your production
- **Building Upgrades:** Click existing buildings to level them up for increased output

## 💾 Save System

- **Auto-save:** Game state is saved automatically to localStorage
- **Manual save:** Use the Save button in the controls bar
- **Portable save codes:** Export/import your progress as a text code to transfer between devices

## 🎨 Tech Stack

- **Single HTML file** — no build step, no dependencies
- **Vanilla JavaScript** (ES5 compatible)
- **HTML5 Canvas** for all game rendering
- **CSS** for UI panels and controls
- **localStorage** for save persistence
