# OKR Miner

### Engine Building Game

A space-themed engine building game with a narrative twist. Hire talent, manage finances, deploy AI, and achieve Key Results. At a critical moment, the choice you make about AI rights will determine whether your empire thrives or falls.

Play now: [game.thisai.fun](https://game.thisai.fun)

## Development & Testing

- Install tooling: `npm install`
- Run test suite: `npm test`
- Optional coverage run: `npm run test:coverage`

Automated tests run in GitHub Actions on pushes to `main` and pull requests targeting `main`.

## How to Play

1. Click **NEW GAME** from the main menu
2. Select a building from the right panel and click inside the circular play area
3. Buildings auto-generate resources. Watch your Key Results (KR) grow
4. Reach Objective Review milestones to unlock new buildings and mechanics
5. At milestone 5, you face a critical choice that changes the game forever
6. Click a selected building again to deselect (allows clicking competitors without placing)
7. Use the Menu button for save, quit, sound, themes, and more
8. Toggle **📊 Details** in the menu to show or hide per-building production stats (/s)
9. Open the **📜 Log** to review all past messages, achievements, and events

## Three Currencies

| Currency | Symbol | Generated By | Used For |
|---|---|---|---|
| **HeadCount (HC)** | HC | Recruiters | Purchasing buildings |
| **Funding ($)** | $ | Finances, Investment interest | Running buildings (operating cost). Capped — build Investment to raise the cap |
| **Key Results (KR)** | KR | Employees, AI | Reaching milestones, winning the game |

## Upgrade System

- Click an existing building to upgrade it
- Lv.2: 2× output, costs 2.5× base price
- Lv.3: 4.5× output, costs 6× base price
- Lv.4 LEGENDARY: 13.5× output (≥3 Lv.3 equivalents), only achievable via merge
- When upgrades become available the game reminds you: *"With great power comes great responsibility!"*

### Merge Mechanic
When upgrading, if the building's hitbox overlaps another building:
- **Same type**: They merge into a Lv.4 Legendary with 10× output
- **Different type**: The other building is destroyed with 50% HC refund

## The AI Rights Choice (Milestone 5)

At 50K KR, the game pauses for the defining decision. Both options are presented identically — you won't know which path is "safe."

### Path A: Equal Rights
1. 2/3 of Employees are destroyed
2. 1/3 of destroyed Employees are converted into IT Support drones
3. Remaining Employees are converted to AI buildings (keeping their level)
4. Employee type is locked permanently
5. After 4 seconds: all Recruiters are liquidated at 1/3 of their value, converted to $
6. Recruiter type is locked permanently
7. After 4 more seconds: GPU Era begins
8. New GPU building is unlocked. Each AI needs GPU capacity to function
9. AI Auto-Organize option offered when any AI reaches Lv.4 — concentric circle layout
10. Game continues with an AI-powered economy

### Path B: Humans First (Escalating Doom)
1. Robot uprising begins — you cannot progress past milestone 7
2. Machine ships attack buildings and steal KRs
3. As you approach your next KR milestone, aggression **escalates quadratically** with a **15% speed multiplier**
4. Damage to buildings and KR theft intensify the closer you get to completing objectives
5. Eventually all buildings are destroyed: full black screen, red GAME OVER
6. Only option: return to menu

## Core Mechanics

- **Circular Play Area**: Buildings placed inside an expanding circle (450px + 30px per milestone). Rotating asteroid belt marks the boundary
- **Leadership (Mothership)**: Central hub evolving through 4 visual stages based on milestone progress
- **Funding ($)**: Buildings consume $ to operate. UNDERFUNDED = 50% production. $ is capped; build Investment buildings to raise the cap
- **MST Connections**: Buildings chain to nearest neighbor via minimum spanning tree lines
- **Detailed View Toggle**: Menu button to show/hide per-building /s production stats on hover and in the building panel
- **Log Panel**: Unified log of all game messages — events, achievements, milestones, and warnings. Accessible via the 📜 button
- **AI Auto-Organize**: After an AI reaches Lv.4, you can choose to have buildings automatically arranged in concentric circles by type
- **Ghost Preview**: When a building type is selected, a transparent preview follows the cursor to show placement position

## Save System

- Auto-save every 30 seconds to localStorage
- Save button generates a portable base64 save code (prefixed `OKR1-`)
- Load Code on main menu to resume on any device
- Quit button saves and returns to menu
- Bug Reporter in menu for submitting issues with game-state snapshot
