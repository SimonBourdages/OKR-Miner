# OKR Miner

### Engine Building Game

A space-themed engine building game with a narrative twist. Hire talent, manage finances, deploy AI, and achieve Key Results. At a critical moment, the choice you make about AI rights will determine whether your empire thrives or falls.

Play now: [game.thisai.fun](https://game.thisai.fun)

---

## How to Play

1. Click **NEW GAME** from the main menu
2. Select a building from the right panel and click inside the circular play area
3. Buildings auto-generate resources. Watch your Key Results (KR) grow
4. Reach Objective Review milestones to unlock new buildings and mechanics
5. At milestone 5, you face a critical choice that changes the game forever
6. Click a selected building again to deselect (allows clicking competitors without placing)
7. Use the Menu button for save, quit, sound, and more

## Three Currencies

| Currency | Symbol | Generated By | Used For |
|---|---|---|---|
| **HeadCount (HC)** | HC | Recruiters | Purchasing buildings |
| **Funding ($)** | $ | Finances, Investment interest | Running buildings (operating cost) |
| **Key Results (KR)** | KR | Employees, AI | Reaching milestones, winning the game |

## Upgrade System

- Click an existing building to upgrade it
- Lv.2: 2x output, costs 2.5x base price
- Lv.3: 4.5x output, costs 6x base price
- Lv.4 LEGENDARY: 10x output, only achievable via merge

### Merge Mechanic
When upgrading, if the building's hitbox overlaps another building:
- **Same type**: They merge into a Lv.4 Legendary with 10x output
- **Different type**: The other building is destroyed with 50% HC refund

## The AI Rights Choice (Milestone 5)

At 50K KR, the game pauses for the defining decision:

### Path A: Equal Rights (Recommended)
1. 2/3 of Employees are removed from the field
2. Remaining 1/3 are converted to AI buildings (keeping their level)
3. Employee type is locked permanently
4. After 4 seconds: all Recruiters are liquidated at 1/3 of their value, converted to $
5. Recruiter type is locked permanently
6. After 4 more seconds: GPU Era begins
7. New GPU building is unlocked. Each AI needs GPU capacity to function
8. Game continues with an AI-powered economy

### Path B: Humans First (Game Over)
1. Robot uprising begins
2. Machine ships progressively attack buildings and steal KRs
3. Attacks accelerate over time until all buildings are destroyed
4. When everything is gone: full black screen, red GAME OVER
5. Only option: return to menu

## Core Mechanics

- **Circular Play Area**: Buildings placed inside an expanding circle. Asteroid belt marks the boundary
- **Leadership (Mothership)**: Central hub evolving through 4 visual stages
- **Funding ($)**: Buildings consume $ to operate. UNDERFUNDED = 50% production
- **MST Connections**: Buildings chain to nearest neighbor via minimum spanning tree
- **100 Random Events**: Meteor showers, stellar bonuses, cosmic shifts
- **5 Themes**: Emerald Matrix (default), Neon Synthwave, Solar Forge, Arctic Command, Deep Space. Each gives a small production bonus

## Save System

- Auto-save every 30 seconds
- Save button shows a portable base64 save code
- Load Code on main menu to resume on any device
- Quit button saves and returns to menu
